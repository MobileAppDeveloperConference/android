<h2>Tech Stack</h2>

- 100% Kotlin + Jetpack Compose
- Kotlinx Libraries
  - kotlinx.coroutines
  - kotlinx.serialization
  - kotlinx.collections.immutable
- Jetpack
  - Startup
  - ViewModel
  - Navigation
  - Room + ksp
- Dagger-Hilt + ksp
- Timber
